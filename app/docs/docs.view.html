What's inside the box?!

Gulp

Automation. The Gulp file is set up for livereload on SASS, AngularJs, img and angular view chnages.

The Sass stuff...don't worry its only a small file to start out with. Blow it up and put your own mark on it.

You might notice that there are only minified files generated by Gulp. Don't fret, sourcemapping is set up for debuggin and readiblity. Less files, better experience in the browser dev tool.

AngularJS

Inside the MEAN seed is a miny AngularJS app seed to get you rollin

Express

MEAN seed is using the Express framework. Nuff said.

MongoDB

You dont need to use MongoDB, but there are some boiler plate files in here if you want to make your site dynamic, otherwise, ignorem or blow them up.

Get Started

Clone the repo

To get started, clone the repo.

$ git clone https://github.com/maviola5/mean_seed.git

Run install

$ npm install

Fire up Gulp

$ gulp
